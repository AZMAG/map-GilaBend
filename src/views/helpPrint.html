Print page allows you to create a PDF file of the current view of the map. Clicking the Print button takes a snapshot of the current window, incorporating any map type, layers, and zoom level currently selected. Choose your choice of map layout using the the drop-down menu. When the printing process is done your map will be displayed in a new browser window or browser tab. From this tab or window you can save the PDF document to a new location.

If page does not automatically load you may have to check your browsers pop-up blocker.

There is no requirement to have any Adobe software installed in order to generate the PDF file.
