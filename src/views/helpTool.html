The measurement tool provides tools for measuring both area and distance as well as determining the x,y location.

When the measurement tool is active the pop-ups in the map will be inactivated. To identify a layer using pop-ups make sure the measurement tool is not active.

Measurement Area Tool
  1. Click the measure Area tool button on the Measurement Tools panel.
  2. Select desired measurement units in drop-down on the right.
  3. Use the mouse pointer to draw a polygon representing the area you want to measure.
  4. Click on the map location where you want to start measuring.
  5. Click to draw each new line segment of the polygon.
  6. Double-click to end the polygon.
  7. Results are displayed in the Measurement Results box below the tool bar.
  8. Click the measure Area button on the Measurement Tools panel to clear results and start over.
Measurement Distance Tool
  1. Click the measure Distance tool button on the Measurement Tools panel.
  2. Select desired measurement units in drop-down on the right.
  3. Use the mouse pointer to draw a line representing the distance you want to measure.
  4. Click on the map location where you want to start measuring.
  5. The line can have more than one line segment.
  6. Double-click to end the line.
  7. Results are displayed in the Measurement Results box below the tool bar.
  8. Click the measure Distance button on the Measurement Tools panel to clear results and start over.
Measurement Location Tool
  1. Click the Location tool button on the Measurement Tools panel.
  2. Use the mouse pointer to find the Longitude and Latitude of a location on the map.
  3. As you move the mouse over the map the Longitude and Latitude will appear in the Measurement Results box below the tool bar.
  4. Clicking on the map will add a pin showing the location and final results will be displayed in the Measurement Results box below the tool bar.
  5. Click the Location tool button on the Measurement Tools panel to clear results and start over.

To help with more accurate measurement results you can use the Snapping tool function to snap to the parcels layer line. Do this by pressing the CTRL key and use the mouse to hover near a parcel line. The mouse pointer will snap to a parcel vertex.
